# SySeBa 1.0.0 - Python Legacy

[English](https://github.com/okno/SySeBa/blob/main/docs/releases/v1.0.0-python.md) |
[Indice documentazione](https://github.com/okno/SySeBa/blob/main/docs/README.it.md)

Questa release conserva l'implementazione Python originale usata dalle
installazioni Linux esistenti. È un punto di recupero indipendente e
immutabile mentre SySeBa 2 prosegue come linea C nativa.

## Contenuto

- Servizio backup/restore Python
- Dashboard testuale e log eventi SQLite
- Web UI protetta da token
- Supporto servizio systemd
- Script aggiornamento, snapshot e rollback versione esatta

## Piattaforma e requisiti

- Linux con Python 3.8 o successivo
- `watchdog` e `psutil` da `requirements.txt`
- systemd per l'installazione servizio

Non contiene eseguibili nativi Windows o macOS. Su quelle piattaforme usa
SySeBa 2.0.0.

## Installazione

```bash
tar -xzf syseba-1.0.0-python-linux.tar.gz
cd syseba-1.0.0-python
python3 -m pip install -r requirements.txt
sudo ./syseba-maintenance.sh install
```

Controlla `syseba.conf` prima di avviare. Database, log, token e configurazione
locale sono esclusi intenzionalmente dall'archivio.

critiche di sicurezza o compatibilità. Nuove funzioni e piattaforme riguardano
la linea C nativa.
